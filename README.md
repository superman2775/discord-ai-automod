# discord-ai-automod

How to set up:
1. Clone the repository
2. Create an .env file and fill in the values (.env.example is provided as template)
3. Install dependencies using `pip install -r requirements.txt`
4. Run the bot using `python main.py`

Features:
Your Discord community will be safe from idiots trying to ruin your server! This automod notices every single message that doesn't fit. It looks at chat context and deletes innappropiate messages. You get notified in a custom log channel every time he does that, so you notice what your users are doing!

Made by super_man2775.
The project is not hosted anywhere, so you can't add the bot. You need to host it yourself, with Python 3.12. Have fun!
